图像迷你实验室，可进行图像实验，图像处理，图像分析。  



## 食用指南  
- 项目环境：PyQt5 + Qt Designer + Python 3.7  
- ui编译：`pyuic5 ImageMiniLabUI.ui –o ImageMiniLabUI.py`  
- 包：opencv-python、PyQt5、numpy
